Exclusive Scheduling for LEX Membership Plan Holders

Schedule Your Member Service

As a LEX member you have access to seasonal tune-ups and inspections. Click the floating button in the bottom-right corner, or use the button below.

WordPress Installation

Embed code for your Members page:

<!-- LEX Members Scheduler Widget --> <link rel="stylesheet" href="https://scheduler-mu-three.vercel.app/lex-members-scheduler.css"> <script> window.LEXMembersSchedulerConfig = { apiEndpoint: 'https://scheduler-mu-three.vercel.app/api/lex-booking', autoButton: true, buttonText: 'Schedule Member Service', position: 'bottom-right' }; </script> <script src="https://scheduler-mu-three.vercel.app/lex-members-scheduler.iife.js"></script>

Configuration Options

{ apiEndpoint: '/api/lex-booking', autoButton: true, buttonText: 'Schedule Member Service', position: 'bottom-right', // All the same customization options as the main scheduler: logoUrl: 'https://example.com/your-logo.png', headerColor: '#0e2a4a', buttonColor: '#C9A14A', tagline: 'Exclusive Service for LEX Members', phoneNumber: '(972) 466-1917', // Members-specific defaults (can override): headerTitle: 'Member Scheduling', headerSubtitle: 'Seasonal Tune-Ups • Inspections', step1Heading: 'Which service would you like to schedule?', }

Manual Control

// Open the members scheduler LEXMembersScheduler.open(); // Close it LEXMembersScheduler.close(); // Toggle LEXMembersScheduler.toggle();
